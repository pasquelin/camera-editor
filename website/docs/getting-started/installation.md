# Installation

```bash
pnpm add @media-studio/sdk @media-studio/ui
# Expo : preview/caméra/export natifs
npx expo install @shopify/react-native-skia expo-camera react-native-reanimated
```

Activez la New Architecture (`app.json` : `"newArchEnabled": true`) et montez le
`MediaStudioProvider` à la racine. Voir [Quick start](./quick-start.md).
