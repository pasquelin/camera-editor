# Changesets

Ce dossier gère le versionnage et les changelogs des packages `@media-studio/*`.

- Branche de base : `develop`.
- À chaque modification d'un package, ajouter un changeset : `pnpm changeset`.
- Versionner : `pnpm version-packages`. Publier : `pnpm release`.

Voir [docs/11-MONOREPO.md](../docs/11-MONOREPO.md) et
[docs/ADR/0014-publishing-changesets-npm.md](../docs/ADR/0014-publishing-changesets-npm.md).
